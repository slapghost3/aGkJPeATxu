# Python Django图书管理系统 python310

## 项目概述

本项目是基于Python的Django框架开发的图书管理系统。该系统旨在帮助用户高效管理图书信息，提供便捷的查询、更新和删除图书的功能。

## 技术栈

- 后端：Python Django
- 数据库：SQLite

## 功能模块

- 图书信息管理
 - 图书查询
 - 图书添加
 - 图书修改
 - 图书删除
- 用户管理
 - 用户注册
 - 用户登录
 - 用户权限控制

## 系统角色

- 系统管理员：负责图书信息管理和用户管理
- 普通用户：查询图书，进行基本的图书信息操作

## 运行环境

- Python 3.x
- Django 3.x
- SQLite

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 安装SQLite数据库
4. 克隆项目代码
5. 执行数据库迁移命令
6. 运行开发服务器

## 目录结构

```
project/
│
├── apps/ # 应用目录
│ └── books/ # 图书应用
│
├── db/ # 数据库文件目录
│ └── sqlite.db # SQLite数据库文件
│
├── manage.py # 管理脚本
└── project/ # 项目配置目录
```

## 核心亮点

- 基于成熟的Python Django框架，易于维护和扩展
- 使用SQLite数据库，无需复杂的数据库配置，即可快速上手
- 清晰的目录结构，便于理解和二次开发
- 系统功能全面，覆盖图书管理的基本需求
- 简洁的权限控制，保障系统安全运行
